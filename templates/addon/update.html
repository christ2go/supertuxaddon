Add a new Addon version

Here you can add a new Version of one of your addon. You can upload the addon archive directly, or a git commit from of your repositorys, or import from the supertux-data repository. You then have to select the supertux versions, which your addon can be used with.

Information about the new version
Please note, that the version number has to be unique. (You can't change a once published version)

If your addon was created in classic mode, the version number supplied above must match the version number given in the .nfo file. You can now import the addons version. Please note, that only import your own github repositorys, if you would like to use a repository, which you don't own you have to fork it.

  • Upload via HTTP
  • Import from supertux-data
  • Import from Github
